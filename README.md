- **GitHub User:** `PooyanGhorbani`
- **Repository:** `Pooyan`
- **Branch:** `main`
- **Visibility:** `private`
- **Repository URL:** `https://github.com/PooyanGhorbani/Pooyan`

### وضعیت نصب آسان
چون ریپوی فعلی **private** است، این دستور برای کاربران عمومی کار نمی‌کند:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/PooyanGhorbani/Pooyan/main/pooyan.sh)
```

### روش فعلی برای ریپوی private
1. فایل `pooyan.sh` را مستقیم روی سرور آپلود کنید.
2. این دستورها را اجرا کنید:

```bash
chmod +x pooyan.sh
bash pooyan.sh
```

### اگر بعداً ریپو را public کردی
آن وقت دستور نصب یک‌خطی این خواهد بود:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/PooyanGhorbani/Pooyan/main/pooyan.sh)
```

### نکته
